[博客地址](http://47.104.98.140:3000)

## 项目运行

clone项目源码
```
git clone git@github.com:xiaoxiao1008150/nuxt_blog.git
```
安装依赖
```
cd nuxt_blog
npm install/yarn install
```
运行

```
npm run dev
```
打开浏览器进入localhost:3000,查看效果
